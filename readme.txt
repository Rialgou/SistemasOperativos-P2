Para utilizar el simulador, abrir terminal y ubicarse en la carpeta que contiene el makefile
escribir "make" para compilar y "make run" para compilar y ejecutar el programa (sin comillas)
una vez ejecutado el programa
1. ingresar cantidad de procesos (entero) n
2. ingresar cantidad de hebra especializadas (entero) m
3. ingresar un rango de tiempo de ejecución (dos enteros) a b
luego se ejecutara el programa entregando información sobre que proceso se esta trabajando,
en que runqueue, cuanto tiempo falta para terminar un proceso, prioridad nueva de proceso y 
creación de un nuevo proceso.

OBSERVACIÓN: el tiempo se maneja en segundos
